## 🎯 Быстрый старт

### Для записи видео:

1. **Подготовка:**
   ```bash
   # Запустить систему
   docker-compose up -d
   
   # Проверить статус
   curl http://localhost:9000/api/status
   ```

2. **Настроить терминал:**
   - Шрифт: JetBrains Mono, 18pt
   - Тема: Dark
   - Разрешение: 1920x1080

3. **Открыть шпаргалку:**
   ```bash
   cat docs/VIDEO-CHEATSHEET.md
   ```

4. **Начать запись!**

---

### Для монтажа:

1. **Импортировать материалы:**
   - Запись экрана
   - Голос (если отдельно)
   - Музыка
   - Звуковые эффекты

2. **Следовать гайду:**
   ```bash
   cat docs/VIDEO-EDITING-GUIDE.md
   ```

3. **Применить стиль:**
   - Цвета: #00FF00, #FF0000, #00BFFF
   - Шрифты: Roboto, Fira Code
   - Переходы: Fade, Wipe

4. **Экспортировать:**
   - Формат: MP4 (H.264)
   - Разрешение: 1920x1080
   - FPS: 60
   - Битрейт: 12 Mbps

## 🎨 Визуальный стиль

### Цветовая схема:
- **Фон**: #0C0C0C (чёрный)
- **Текст**: #FFFFFF (белый)
- **Успех**: #00FF00 (зелёный)
- **Ошибка**: #FF0000 (красный)
- **Инфо**: #00BFFF (синий)

### Шрифты:
- **Заголовки**: Roboto Bold, 48pt
- **Текст**: Roboto Regular, 32pt
- **Код**: Fira Code, 18pt
- **Терминал**: JetBrains Mono, 16pt

### Анимации:
- **Появление**: Fade in + Slide (0.3s)
- **Переходы**: Fade (0.5s)
- **Акценты**: Zoom (0.4s)

---

## 🎵 Музыка и звуки

### Музыка:
- **Intro/Outro**: Энергичная, tech (-15dB)
- **Main**: Спокойная, фоновая (-25dB)

### Звуковые эффекты:
- ✅ **Успех**: "Ding" (+0dB)
- ❌ **Ошибка**: "Buzz" (-3dB)
- 🔄 **Переход**: "Whoosh" (-6dB)
- 💡 **Подсказка**: "Pop" (-12dB)

### Голос:
- Громкость: -3dB (пик)
- Компрессия: 3:1
- EQ: +3dB на 2-4kHz
- De-esser: Включён
